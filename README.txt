
# Проект регистрации и входа пользователей

Этот проект представляет собой простую систему регистрации и входа пользователей, реализованную с использованием Node.js, Express и файловой системы для хранения данных. Пользователи могут зарегистрироваться, войти в систему, а также просматривать свой профиль.

## Структура проекта

- `public/` - папка с HTML, CSS и JavaScript файлами для фронтенда.
- `users.json` - файл для хранения данных пользователей.
- `server.js` - серверная часть на Node.js, обрабатывающая запросы.
- `styles.css` - стили для фронтенда.

## Требования

Перед тем как начать, убедитесь, что у вас установлены следующие инструменты:

- [Node.js](https://nodejs.org/) версии 14 и выше.
- [npm](https://www.npmjs.com/) (менеджер пакетов для Node.js), который обычно устанавливается вместе с Node.js.

## Установка и запуск проекта

1. Клонируйте репозиторий на вашу локальную машину:

   ```bash
   git clone https://github.com/ваш-репозиторий/проект.git
   ```

2. Перейдите в директорию проекта:

   ```bash
   cd проект
   ```

3. Установите зависимости, используя npm:

   ```bash
   npm install
   ```

4. Запустите сервер:

   ```bash
   node server.js
   ```

5. Откройте браузер и перейдите по адресу:

   ```
   http://localhost:3000
   ```

   Это откроет страницу с формой для входа в систему.

## Описание функций

- **Регистрация**: На странице регистрации пользователь вводит логин, email, дату рождения, пол, пароль и повторный пароль. Если пароли совпадают, новый пользователь добавляется в файл `users.json`.
- **Вход**: На странице входа пользователь вводит логин и пароль. Если данные верны, он будет перенаправлен в профиль.
- **Профиль**: После успешного входа пользователю показывается его профиль с данными, такими как логин, email, дата рождения и дата регистрации.

## Формат данных в `users.json`

Пример структуры данных:

```json
[
  {
    "username": "user1",
    "email": "user1@example.com",
    "birthdate": "2000-01-01",
    "gender": "male",
    "password": "securepassword",
    "registrationDate": "2024-12-14T10:50:43.888Z"
  },
  {
    "username": "user2",
    "email": "user2@example.com",
    "birthdate": "1995-05-25",
    "gender": "female",
    "password": "anotherpassword",
    "registrationDate": "2024-12-14T10:52:00.123Z"
  }
]
```

## Заметки

- Для хранения паролей в реальном проекте рекомендуется использовать хеширование паролей с помощью библиотеки, такой как bcrypt. Это повысит безопасность вашего приложения.
- Файл `users.json` используется для хранения данных пользователей. В реальной системе, для масштабируемости и безопасности, рекомендуется использовать базу данных.
